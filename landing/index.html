Capabilities

覆盖启动、搜索、阅读、AI 和插件的桌面入口。

Qx 用稳定的三层 Shell 组织常用桌面任务：Top Bar 负责搜索，Main Area 展示结果与内容，Context Panel 放辅助信息，Bottom Island 反馈状态。

⌘

Cardinal 全局文件搜索

接入 Cardinal 搜索索引，支持全局文件检索、路径片段、类型/大小/日期过滤，并和应用、命令、插件结果一起排序。

C

剪贴板历史

持久化文本、链接和图片记录，支持置顶、预览、类型过滤和快捷动作。

R

RSS 阅读器

订阅源、文章列表、详情阅读和后台刷新，在同一套 Shell 中完成。

AI

QxAI

内置多 provider AI 对话，支持流式响应、持久记忆和插件侧 Agent 设置。

P

插件系统

沙箱 iframe 运行时、RPC 桥接、插件市场和 Raycast 扩展转换。

M

宏与工具

录屏、OCR、天气、V2EX、GitHub 贡献图、文本工具和宏录制逐步集成。

Workflow

从呼出到执行，保持一条直线。

Vercel 风格的克制设计适合 Qx：高对比文本、细边框分区、少量动效、明确状态，让用户反复使用时不用重新理解界面。

01

快捷键呼出

菜单栏常驻，全局快捷键打开主搜索，不打断当前工作上下文。

02

搜索定位

应用、文件、插件和模块统一排序，结果区只保留可扫描信息。

03

上下文动作

右侧面板只呈现辅助入口，真实操作进入底栏 Actions 协议。

04

状态反馈

同步、下载、导入和错误进入 Bottom Island，完成后回到轻状态。

Get Qx

下载，放进 Applications，然后用键盘打开。

适用于 macOS 14.0+。可从 GitHub Releases 下载，也可以使用 Homebrew 安装；源码可本地构建，插件和转换器随项目继续迭代。
